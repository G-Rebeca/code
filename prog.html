Linguagens de programação

Pascalzim

O Pascalzim é um ambiente de desenvolvimento integrado e um compilador para a linguagem de programação Pascal desenvolvido no Departamento de Ciências da Computação da Universidade de Brasília concebido para fins educacionais. O mesmo implementa um subconjunto da linguagem Pascal e contém recursos muito utilizados por iniciantes no estudo dessa linguagem.

Python

Python é uma linguagem Open-Source de propósito geral usado bastante em data science, machine learning, desenvolvimento de web, desenvolvimento de aplicativos, automação de scripts, fintechs.
