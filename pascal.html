Tipos primitivos

Tipo Primitivo são os tipos de dados mais simples, isto é, a informação em sua forma mais primitiva.

  • integer para tipos inteiros
  • string para conjunto de caracteres
  • boolean para True or False
  • real para números reais

Estruturas condicionais

Uma Estrutura de Condição, como o próprio nome já diz, verifica a condição dos argumentos passados e, executa um comando caso a condição seja verdadeira.

  • if

    • IF condição THEN
    BEGIN
    comando1;
    comando2;
    comando3;
    END;

  • else

    • IF condição THEN
    BEGIN
    comando1;
    comando2;
    END
    ELSE
    BEGIN
    comando3;
    comando4;
    END;

Estruturas de repetição

É um recurso das linguagens de programação responsável por executar um bloco de código repetidas vezes enquanto determinada condição é atendida.

  • for

    • for contador:=ValorInicial to ValorFinal do
    begin
    comandos;
    end;

  • repeat

    • repeat
    comando1;
    comando2;
    comando3;
    until ;

  • while

    • while do
    begin
    comandos;
    end;
