Tipos primitivos

Tipo Primitivo são os tipos de dados mais simples, isto é, a informação em sua forma mais primitiva.

  • int para tipos inteiros
  • str para conjunto de caracteres
  • bool para True or False
  • float para números reais

Estruturas condicionais

Uma Estrutura de Condição, como o próprio nome já diz, verifica a condição dos argumentos passados e, executa um comando caso a condição seja verdadeira.

  • if
  • elif
  • else

    • if soma > 0:
    print "Maior que Zero."
    elif soma = 0:
    print "Igual a Zero."
    else:
    print "Menor que Zero."

Estruturas de repetição

É um recurso das linguagens de programação responsável por executar um bloco de código repetidas vezes enquanto determinada condição é atendida.

  • for

    • fruits = ["apple", "banana", "cherry"]
    for x in fruits:
    print(x)

  • while

    • contador = 0
    while (contador < 5):
    print(contador)
    contador = contador + 1
